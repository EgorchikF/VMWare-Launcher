# 🚀 VMware Simple Launcher

**[🌍English Version(Английская версия)](https://github.com/EgorchikF/VMWare-Launcher/blob/main/README.md)**

Минималистичный фоновый лаунчер для **VMware Workstation**.

### ✨ Возможности
- Автоматически запускает все необходимые службы VMware
- Запускает VMware Workstation
- Следит за процессом программы в фоне
- При закрытии VMware:
  - Завершает все оставшиеся процессы VMware
  - Переводит службы в состояние **Отключено**
- Полностью без интерфейса — нет окон и консоли
- Написан на **C# .NET 8**

### Зачем это нужно?
Службы VMware продолжают «жрать» оперативную память и процессор даже после закрытия программы. Этот лаунчер решает проблему, запуская службы только на время работы.

### 📋 Как пользоваться
1. Соберите проект в конфигурации **Release**
2. Положите `.exe` файл в удобное место
3. Готово!

### 🛠 Требования
- Windows 10 / 11 (64-bit)
- VMware Workstation Pro (16.x и выше)
- .NET 8.0 Desktop Runtime
